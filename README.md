# A Comprehensive Measure of Well-Being

## Project Description

This project predicts and displays the Human Development Index (HDI) of different countries based on key development indicators such as Life Expectancy, Mean Years of Schooling, Expected Years of Schooling, and GNI per Capita. It provides a simple and interactive web interface for users to explore a country's well-being.

## Features

- Country-wise HDI prediction
- Displays Life Expectancy
- Displays Mean Years of Schooling
- Displays Expected Years of Schooling
- Displays GNI per Capita
- Displays HDI Score
- Displays HDI Category
- Clean and responsive single-page interface

## How to Run

1. Install the required packages:
   pip install -r requirements.txt

2. Run the Flask application:
   python app.py

3. Open the browser and visit:
   http://127.0.0.1:5000/
